## How to Run Locally

1. Clone the repository:

```bash
git clone git@github.com:andriyParashchuk/wallet-app.git
cd wallet-app
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

4. Open in browser:

```
http://localhost:5173
```
